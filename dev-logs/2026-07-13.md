### 运维保障

#### 4. Supabase 保活定时任务 🆕
- **文件**：[.github/workflows/keep-alive.yml](../.github/workflows/keep-alive.yml)
- **问题**：Supabase 免费项目 7 天不活跃会被暂停，暑假/寒假期间无人使用会触发
- **方案**：GitHub Actions 每周 cron 定时 ping Supabase REST API
- **原理**：每周日 UTC 0:00 自动发一条 `GET /rest/v1/members?select=id&limit=1` → Supabase 检测到活动 → 不活跃计时器归零
- **状态**：已 push 到 master 分支，GitHub Actions 面板可见
