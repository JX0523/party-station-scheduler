- [ ] 1.1 使用 Vite 初始化 React 项目
- [ ] 1.2 安装依赖：react-router-dom、@supabase/supabase-js、xlsx
- [ ] 1.3 创建目录结构（components/ pages/ hooks/ lib/ styles/）
- [ ] 1.4 创建 Supabase 项目，获取 API Key
- [ ] 1.5 在 Supabase SQL Editor 中执行建表语句
- [ ] 1.6 配置 RLS 安全策略
- [ ] 1.7 创建 supabase.js 客户端配置文件
- [ ] 1.8 实现登录页面（Supabase Auth）
- [ ] 1.9 创建全局布局组件（导航栏 + 内容区）
- [ ] 1.10 配置路由（各页面占位）
