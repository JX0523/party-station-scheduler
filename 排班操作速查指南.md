## 二、每周例行操作（5分钟搞定）

这是你**每周都要做**的事情，按顺序来：

### 周日晚或周一早上（排本周班）

```
步骤 1 ── 打开首页，确认顶部显示的"第 X 周"是否正确
步骤 2 ── 看一眼本周有没有调休/放假（见下方"场景A"）
步骤 3 ── 有人请假？→ 点「请假」→ 选替补（见"场景B"）
步骤 4 ── 检查排班表，确认每天每时段都有人
步骤 5 ── 完成！
```

### 每周结束后（锁定本周 + 开启下周）

```
步骤 1 ── 点左侧导航 → 「学期设置」
步骤 2 ── 把「当前是第几周」的数字 +1（比如5改成6）
步骤 3 ── 保存
步骤 4 ── 回到首页 → 系统自动生成新一周排班
```

> ⚠️ **这一步很重要！** 不更新的话，下周排班无法锁定，历史数据会乱。

## 三、常见场景 → 怎么操作

### 🟢 场景A：遇到放假 / 调休

**情况1：某天放假（如周三清明节）**

1. 在排班表上方找到「周三」按钮（绿色）
2. 点一下 → 变灰色 = 当天不排班
3. 点「🔄 重新生成」排班

**情况2：周末调休上班（如周六补周一课）**

1. 点「周六」按钮 → 变橙色，下方会自动弹出两个下拉框：
   - 🔵 **单周补周X**：选「周一」（单周时按周一课表检查）
   - 🟠 **双周补周X**：选「周一」（双周时也按周一课表检查）
2. 如果单双周补的课不一样，分别选就行
3. 确认周六时段人数（默认0，需要手动改成1）
4. 点「🔄 重新生成」

> 💡 按钮颜色速记：🟢绿色=上班 | ⚫灰色=放假 | 🟠橙色=调休上班

---

### 🟡 场景B：有人请假 → 找替补

1. 在排班表中找到那个人，点旁边的 `请假` 按钮
2. 弹窗确认 → 点确定
3. 系统自动弹出**替补候选列表**，注意看「课表」列：
   - 🟢 **空课表 / ✅ 优先**（绿色高亮）→ 这人完全没课表，随时能来，首选！
   - ⚪ **该时段空闲** → 这个时段没课，次选
4. 选一个人，点「选为替补」→ 完成
5. 被请假的人这周不计时长，系统下周会自动补排他

> ⚠️ 如果弹窗显示"暂无可替换的候选人"，说明这个时段所有人都没空。可以试试让其他时段的人换过来。

---

### 🔵 场景C：临时需要多加一个人

1. 找到对应时段格子，点 `+ 加人`
2. 同样优先选 🟢空课表 的同学
3. 点「选为替补」即可

---

### 🟣 场景D：某时段缺人（显示"⚠ 还缺 X 人"）

说明这个时段人不够，可以：

- **方案1**：去「时段配置」页面，把那个时段的需求人数改大，重新生成
- **方案2**：手动点 `+ 加人` 补上
- **方案3**：如果实在没人有空，就让它空着（系统允许）

## 四、开学初要做的事（一学期一次）

新学期开始，需要重新配置以下内容（找站长一起搞）：

| 顺序 | 页面 | 做什么 | 预计时间 |
|:--:|------|------|:--:|
| 1 | 学期设置 | 改学期名、总周数、当前周设为1、设置首周单/双周 | 2分钟 |
| 2 | 成员管理 | 删掉毕业生、添加新生、确认名单正确 | 15分钟 |
| 3 | 课表录入 | 让所有人录课表（或你统一录），每人要录**单周+双周**两份 | 1-2小时 |
| 4 | 时段配置 | 确认周一至周五每时段1人，周末0人 | 2分钟 |
| 5 | 首页 | 点重新生成，完成第1周排班 | 10秒 |

## 五、关键规则速记

| 规则 | 一句话解释 |
|------|-----------|
| 谁优先排？ | 部员 > 部长 > 主席团（部员是主力） |
| 课表冲突？ | 有课 = 不能排；没课 = 可以排 |
| 连续值班？ | 同一个人不会连续两周排上 |
| 公平性 | 排得少的优先排 |
| 补排 | 请假后下周自动优先安排 |
| 调休 | 周末加班用对应工作日的课表检查冲突 |

## 六、常见问题 FAQ

**Q：排班表里出现"未安排"怎么办？**
A：说明这个时段有空的人不够。检查时段配置是否需要调整，或手动加人。

**Q：锁定的周还能改吗？**
A：不能。当前周之前的周自动锁定，只能看不能改。这是为了防止数据被乱改。

**Q：为什么主席团几乎不排班？**
A：正常设计。主席团只有全部员+部长都不够5人时才会启用，平时不参与排班。

**Q：忘记更新"当前周"了怎么办？**
A：直接去学期设置改成正确的数字就行。但已经锁定的周不会自动解锁。

**Q：换了电脑还能用吗？**
A：能。数据在云端，任何电脑浏览器打开网址登录即可。

**Q：手机能操作吗？**
A：能打开，但排版不太好。建议用电脑操作。

**Q：误操作了怎么办？**
A：排班可以随时点「🔄 重新生成」，会覆盖当前周的排班。历史周（锁定周）无法修改。
